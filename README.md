# 📊 NSE Bhavcopy Downloader

A simple and powerful tool to **automatically download NSE Bhavcopy data** from the National Stock Exchange (NSE) website.

This application helps traders, analysts, and developers quickly download **historical NSE stock market data** without manually visiting the NSE website.

---

## 🚀 Features

- Automatic download of **NSE Bhavcopy data**
- Supports **historical data download**
- Simple and easy-to-use interface
- No manual browsing of NSE website required
- Fast bulk data download

---

## 📦 Available Versions

### Version 1.0

Supports downloading Bhavcopy data from:

**01 January 2016 → 05 July 2024**

Download file:Bhavcopy_Jan_2016_Till_05_Jul_2024_V1.0.exe


---

### Version 2.1 (Latest)

Supports downloading Bhavcopy data from:

**01 January 2020 → Present**

Due to the file size (**119 MB**), the executable is split into two parts.

Download both files:Bhavcopy_V2.1.z01, Bhavcopy_V2.1.zip


After downloading both files, extract them together to obtain the executable.

---

## ⚙️ How to Use

1. Download the required version.
2. Download both split files if using Version 2.1.
3. Extract the ZIP file.
4. Run the executable file.
5. Select the required date range.
6. Start downloading Bhavcopy data.
7. For more information check HELP.txt file.

---
